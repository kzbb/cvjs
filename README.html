組み立て方

3Dプリンタで必要パーツを出力します。

左上 爪、右上 スキャンボディ_16mm用、下 スキャンベース

ボディは、使用するフィルムに対応したものを出力してください。

爪をベース下面より、突き刺します

ギリギリまで押し込んでください

スプールを組み立てます。

スプールにベアリングをはめ込みます。(きつめに設計してあります。)
ボルトを通し、六角ナットをきつめに締めます。ナットはネジ部分終端で止まります。

断面図

スプールには上下の向きがあります。爪があるほうが上です。
向きが逆だと、操作中にスプールが落下する可能性があります。

ベアリングは爪の無い方から挿入してください。

4つ作ります

ボディにねじ込みます

ネジはボディの下面よりも飛び出ないように
(スマホに傷がつく可能性があります)

ベースと組み合わせます

はめ込みます

爪が開いていれば正しく装着されています。
爪が開かない場合は爪をさらに押し込んでください。

フィルムサイズを変更する際は、この爪をつまんで取り外すことができます。

レンズのとりつけ

レンズは載せるだけですが、不安な場合は両面テープを少し使用しても良いです。

装置部分が完成しました
